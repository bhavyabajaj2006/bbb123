Question 1:
Corrige les phrases:
1. Je veux que vous venez.
...............................................................................................
2. Il faut que vous m'apportez les livres.
...............................................................................................
3. Nous voulons que vous travaillez.
...............................................................................................
4. Je suis contente que vous êtes en vacances.
...............................................................................................
5. Je souhaite que Paul n'est pas absent.
...............................................................................................
ANSWER:
(Use 'subjonctif' in place of the present tense.)
1. Je veux que vous veniez.
2. Il faut que vous m'apportiez les livres.
3. Nous voulons que vous travailliez.
4. Je suis contente que vous soyez en vacances.
5. Je souhaite que Paul ne soit pas absent.

Question 2:
Refais les phrases en utilisant «Il faut que» / « Il est nécessaire que »:
1. Pauline doit payer les frais de scolarité avant le 15 avril.
...............................................................................................
2. Nous devons finir le devoir avant midi.
...............................................................................................
3. Vous devez prendre un taxi pour arriver à l'heure.
...............................................................................................
4. Les parents d'Akanksha doivent rentrer immédiatement.
...............................................................................................
5. Les étudiants doivent rencontrer le professeur de chimie.
...............................................................................................
ANSWER:
(In place of verb 'devoir' use the given expressions.)
1. Il faut que Pauline paie les frais de scolarité avant le 15 avril.
2. Il faut que nous finissions le devoir avant midi.
3. Il faut que vous preniez un taxi pour arriver à l'heure.
4. Il est nécessaire que les parents d'Akanksha rentrent immédiatement.
5. Il est nécessaire que les étudiants rencontrent le professeur de chimie.

Question 4:
Retrouve la phrase:
1. beaucoup / les / il / policiers / aime / lire / romans.
2. ensemble / il / planète / faut / sauver / travailler / pour / la.
3. fière / travail / elle / son / est / de / très.
4. souvent / taxi / il / au / aller/ un / prend / pour / bureau.
5. mis / vase / dans / elle / fleurs /elle / un / achetées / les / a /qu'/a.
ANSWER:
(Formation of the sentence, "Subject + Verb + Object".)
Il aime beaucoup lire les romans policiers.
Il faut travailler ensemble pour sauver la planète.
Elle est très fière de son travail.
Il prend souvent un taxi pour aller au bureau.
Elle a mis les fleurs qu'elle a achetées dans un vase.

Question 5:
Corrige les fautes:
1. Akanksha : Tu parles de quelle émission ? Celui d'hier ?
Denis : Non. Celles que nous avons vu il y a deux jours.
2. Messieurs, mesdames, je vous présente deux marques des machines à laver.
Celles-ci est un IFB, celles-là est un Whirlpool.
ANSWER:
(Use the correct demonstrative pronouns.)
1. Akanksha : Tu parles de quelle émission ? Celle d'hier ?

Denis : Non. Celle que nous avons vue il y a deux jours.
2. Messieurs, mesdames, je vous présente deux marques des machines à laver.
Celle-ci est un IFB, celle-là est un Whirlpool.
